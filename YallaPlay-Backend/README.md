# YallaPlay Backend

Express backend for YallaPlay.

## Run

```bash
npm install
npm run dev
```

Server runs on:

```txt
http://localhost:5000
```

## Routes

- `GET /fields`
- `GET /fields/search?q=nablus`
- `GET /fields/:id`
- `POST /bookings`
- `GET /bookings`
- `PUT /bookings/:id`
- `DELETE /bookings/:id`
- `GET /profile/:userId`
- `PUT /profile/:userId`
