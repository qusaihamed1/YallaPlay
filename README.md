# YallaPlay

YallaPlay is a mobile application developed to simplify sports field reservations in the West Bank, Palestine. The application allows users to browse available football and basketball fields, view field details, check availability, and reserve time slots easily and efficiently.

## Important Note

The final submitted version of this project is available in the following branch:

### `Qusai-Finish-project-lastSubmission`

Please note that this branch contains the latest stable and fully updated version of the project and should be used for evaluation instead of the `main` branch.

---

# Overview

The application focuses on providing a simple and intuitive user experience for sports field reservations. It reduces the effort required to search for available fields and allows users to complete bookings in a structured and efficient way.

---

# Features

- Browse available football and basketball fields
- View field details such as:
  - Price
  - Distance
  - Availability
- Filter fields by sport type
- User authentication:
  - Sign Up
  - Login
  - Forgot Password
- Modern and user-friendly UI/UX
- Offline support using SQLite
- Secure authentication using SecureStore
- Favorites system using AsyncStorage
