YallaPlay is a mobile application developed to simplify the process of reserving football fields in the West Bank, Palestine. The application enables users to browse available fields, view details, and book time slots easily and efficiently.

Overview :

The application focuses on providing a simple and intuitive user experience for sports field reservations. It reduces the effort required to find available fields and allows users to complete bookings in a structured and efficient way.

Features :
Browse available football and basketball fields
View field details such as price, distance, and availability
Filter fields by sport type
Simple and user-friendly interface
User authentication (Sign Up, Login, Forgot Password)
